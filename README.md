# 🦅 RAVEN SYSTEMS — Official Portfolio

Welcome to the official source code for the **Raven Systems** portfolio website. This is a high-performance, security-focused, single-page application showcasing enterprise-grade systems, local-first AI architecture, and full-stack development capabilities.

---

## ✨ Features

- **Zero-Dependency:** Built with pure HTML, CSS, and Vanilla JavaScript. No heavy frameworks or build tools.
- **Performance Optimized:** Inline SVG noise generation, `font-display: swap`, and minimal DOM manipulation.
- **Accessibility First:** Full keyboard navigation support, `aria-labels`, and `:focus-visible` states.
- **Responsive Design:** Fluid typography (`clamp`) and CSS Grid for seamless mobile-to-desktop scaling.
- **Dynamic Rendering:** Project data is injected via a clean, maintainable JavaScript array.
- **Core Competencies Section:** Showcases deep technical skills (Linux, Reverse Engineering, Zero-Trust) without requiring live links.
- **Local/LAN Systems:** Professional handling of internal systems with "Internal Only" and "Private Repo" placeholders.
